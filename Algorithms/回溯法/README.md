# 回溯法
## 思想
回溯法的本质就是<b>一棵决策树(n叉树)的前后序的遍历过程</b>。详细了解可点击[这里](https://zhuanlan.zhihu.com/p/93530380)。

## 做法
简单来说，主要步骤分为if——for——if。<br>
1、第一个if判断遍历结束的条件；<br>
2、第一个for遍历选择列表，第二个if判断该选择是否合法，不合法直接contiue。合法就进行选择，选择完后下一步进行决策；
3、当遍历结果再退回到当前决策，下一步再进行撤回选择。
<br>
伪代码如下：
```python
for 选择 in 选择列表:
    # 做选择
    将该选择从选择列表移除
    路径.add(选择)
    backtrack(路径, 选择列表)
    # 撤销选择
    路径.remove(选择)
    将该选择再加入选择列表
```

# 网格类型的DFS遍历
网格类型题目的要求一般是在`m×n`的网格中进行枚举。若是求最短路径则考虑BFS，求最大收益等是DFS。
## 做法
1、定义方向数组作为游走边界条件；<br>
2、定义一个变量维护最大化收益；<br>
3、根据方向数组做出选择，判断选择是否满足边界条件或者其他条件。<br>
- 满足则定义访问数组记录访问过的记录，置空该记录再进行遍历；<br>
- 不满足则恢复该记录。
